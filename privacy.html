隱私權政策 (Privacy Policy)

生效日期：2026 年 2 月 18 日

1. 簡介

歡迎使用「鴻海集團差旅平台 填表助手」（以下簡稱「本擴充功能」）。本擴充功能致力於保護使用者的隱私權。本政策旨在說明我們如何處理您的資料。

2. 資料收集與使用

我們不會收集、傳輸或分享您的任何個人資料。

本擴充功能的所有運作皆在您的瀏覽器本機端進行。您儲存的填寫範本（包含專案代碼、聯絡人資訊等）僅會儲存在您個人的瀏覽器儲存空間 (`chrome.storage`) 中，用於自動填寫表單，開發者無法存取這些資料。

3. 權限說明

  • Scripting / ActiveTab：僅用於將自動填寫腳本注入至 TERS 網站。
  • Storage：僅用於在您的裝置上保存您的個人範本設定。
  • Host Permissions：僅限於存取 ters.efoxconn.com，確保不會讀取其他網站資訊。

4. 第三方服務

本擴充功能不包含任何第三方追蹤分析工具（如 Google Analytics）或廣告服務。

5. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過 GitHub Issues 聯繫開發者。
